## 评估指标

### 检测率计算
- **真阳性 (TP)**: 实际存在且被检测到的漏洞
- **假阳性 (FP)**: 被检测但实际不存在的漏洞
- **假阴性 (FN)**: 存在但未被检测到的漏洞

```
检测率 = TP / (TP + FN)
精确率 = TP / (TP + FP)
F1分数 = 2 * (检测率 * 精确率) / (检测率 + 精确率)
```

### 当前评估结果

**dvcp.c**:
- Level 1 检测率: ~80% (整数溢出、除零、内存泄漏已检测)
- Level 2 检测率: ~40% (双重释放、UAF未直接检测)

**test_vulnerable.c**:
- Level 1 检测率: ~90% (大部分明显问题已检测)
- Level 2 检测率: ~50% (需要AI分析控制流的漏洞)
