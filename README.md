# 🚀 AI 技术学习项目

> 面向网络安全工程师的 AI 技术实践学习路径

## 📋 项目目标

通过问题驱动的方式，系统学习以下能力：
- ✅ 本地模型运行与调用
- ✅ 模型微调与后训练
- ✅ Agent、MCP、Skill 开发
- ✅ Dify 工作流编排与 A2A 多 Agent 协作

## 🎯 学习路线

| 阶段 | 主题 | 周期 | 核心产出 |
|------|------|------|----------|
| 一 | 基础认知与环境搭建 | 1-2周 | 本地模型对话程序 |
| 二 | 模型微调与后训练 | 3-4周 | 安全领域微调模型 |
| 三 | Agent/Dify/A2A开发 | 6-8周 | 自动化安全Agent |

## 🚦 快速开始

1. 阅读 [完整学习计划](docs/learning_plan.md)
2. 按顺序完成各阶段实验指导
3. 在 `progress.md` 中记录学习进度
4. 实验代码保存在 `experiments/` 目录
